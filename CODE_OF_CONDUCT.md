# Code of Conduct

## Commitment

This project welcomes thoughtful participation regardless of experience,
identity, background, nationality, appearance, disability, religion, or
technical preference. The standard is simple: make the work and the space
safer, clearer, and more useful for the next contributor.

## Expected behavior

- Discuss the work, not the worth of the person doing it.
- Give specific, actionable feedback and receive correction in good faith.
- Respect privacy, attribution, boundaries, and the project’s evidence rules.
- Make room for questions while keeping claims technically accountable.
- Stop an interaction when someone reasonably asks for space.

Harassment, threats, discriminatory language, sexualized attention, deliberate
misinformation, doxxing, and sustained disruption are not accepted.

## Scope and enforcement

This standard applies in issues, pull requests, reviews, discussions, and
project-related public communication.
